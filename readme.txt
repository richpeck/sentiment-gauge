Sentiment Gauge shortcode for Wordpress websites.

== Description ==
Adds our \"sentiment gauge\" shortcode for use in the Wordpress front-end, designed and developed in house.

== Installation ==
Install the plugin and then use the [sentiment-gauge] shortcode. Details provided in the settings page.

== Frequently Asked Questions ==
1. How to use the tool?

Simply install and activate the plugin, then use the [sentiment-gauge] shortcode in your posts or pages.
